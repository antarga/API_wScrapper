# API with Web Scraper

Supplementary code for my thesis titled "Sentence Classification with Hierarchical Neural Networks for Rhetorical Sections Extraction", developed as part of the MSc program "Data Science and Information Technologies" at NKUA.

## Overview

Built with Flask and BeautifulSoup, this tool fetches abstracts and classifies their sentences into 5 or 7 rhetorical categories.
